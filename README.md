the apps functions are explained by the builtin tutorial

what the app can do:
    it can convert any image into ascii in full color or just monochrome
    it can export any converted image in full color (html) or in monochrome (txt)
    it can even enhance the ascii image using the ultra detail mode that uses a lot more characters and can scale the image 2x or 4x for more detailed image

problems with the app:
    it can get slow when using 2x or 4x ultra detail mode (I tried but could not make it faster sorry)
